# Personal LinkedIn Manager AI

A Streamlit-based AI tool that helps manage LinkedIn content using LLMs and recruiter psychology.

-> Features
- Decide whether to post today
- Generate LinkedIn posts
- Write posts about projects you build
- Optimize existing LinkedIn posts
- Suggest best posting time and hashtags

-> Tech Stack
- Python
- Streamlit
- OpenAI API
- Prompt Engineering

-> How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
